# AA-SISTEMA

Projeto experimental de um sistema antiaéreo automatizado utilizando Arduino.

O sistema utiliza um sensor ultrassônico montado em servos para realizar varredura do ambiente e detectar objetos. Quando um alvo é identificado, o sistema realiza tracking automático e aponta um laser para a posição detectada.

## Tecnologias utilizadas

- Arduino
- C++
- Servo Motors
- Sensor Ultrassônico HC-SR04
- Laser Module

## Funcionalidades da V1.0

- Radar de varredura 180°
- Detecção de objetos por distância
- Tracking automático de alvo
- Sistema de predição de movimento
- Mira com laser

## Objetivo do projeto

Estudo de sistemas de detecção, automação e controle utilizando Arduino.
